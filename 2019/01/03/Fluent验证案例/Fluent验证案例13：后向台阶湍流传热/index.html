Fluent验证案例13：后向台阶湍流传热

发表于 2019-01-03 | 分类于 Fluent验证案例 | 阅读次数：

本案例对2D后向台阶的流体流动和对流换热进行了数值模拟。突扩几何导致的流动分离及边界层再附会影响到其后方恒定热流表面的局部换热。本案例利用测量的热流表面的局部Nusselt数分布验证CFD仿真结果。

本案例计算模型如下图所示（图中H=1 m）。采用非均匀四边形网格划分，采用稳态计算，不考虑流体的可压缩性。计算域入口为充分发展流动，其速度采用Profile文件进行指定，确保入口雷诺数为28000，计算中采用RNG k-epsilon湍流模型，并使用标准壁面函数处理湍流。

采用的介质物性参数：

  • Density = 1 kg/m3
  • Viscosity = 0.0001 kg/m-s
  • Conductivity = 1.408 W/m-K
  • Specific Heat = 10,000 J/kg-K

壁面热通量 Q= 1000 W/m2。

1 Fluent设置

  • 以2D、Double Precision模式启动Fluent
  • 利用菜单File → Read → Case…读取文件VMFL013.CAS

1.1 Models设置

  • 激活能量方程
  • 激活RNG k-epsilon湍流模型

1.2 Materials设置

  • 修改材料air的属性参数，如下图所示。

1.3 Profile文件

本案例利用Profile文件定义边界参数。

  • 利用菜单File → Read → Profile…读取文件VMFL013_step_ve.set.prof

1.4 Boundary Conditions

1、velocity-inlet-6

  • 设置边界velocity-inlet-6的入口速度及湍流条件，如下图所示

  • 设置入口温度为273 k

  • 此时可以查看profile定义的入口速度分布，如下图所示

2、pressure-outlet-7

  • 设置出口边界pressure-outlet-7的出口表压为0 pa，湍动能为0.001 m2/s2，湍流耗散率为0.01 m2/s3

  • 切换至Thermal标签页，设置Backflow Total Temperature为273 k

3、wall-4

  • 鼠标双击模型树节点Boundary Conditions > wall-4，弹出对话框中切换至Thermal标签页，设置Heat Flux为1000 w/m2

1.5 Surface Report Definition

监测边界wall-4的平均nusselt数。

  • 鼠标双击模型树节点Solution > Report Definitions，弹出设置对话框中按下图所示进行设置

1.6 Initialization

  • 右键选择模型树节点Initialziation，点击弹出菜单项Initialize进行初始化

1.7 Run Calculation

  • 鼠标双击模型树节点Run Calculation，右侧面板设置Number of Iterations为650
  • 点击按钮Calculate开始计算

2 计算结果

  • 查看速度分布

  • 绘制wall-4边界上nusselt数分布曲线

  • 计算值与实验值对比
